# Number Guessing Game

A fun and interactive number guessing game where the player tries to guess a randomly generated number within a limited number of attempts.

## How to Play

1. The program generates a random number within a defined range (e.g., 1-100)
2. The player enters their guess
3. The program provides hints: "Too High!", "Too Low!", or "Correct!"
4. The player wins by guessing the correct number within the allowed attempts
5. The game tracks the number of attempts used

## Features

- Random number generation
- Configurable number range
- Limited attempts with feedback
- Win/lose conditions
- Play again option

## Getting Started

```bash
git clone https://github.com/Icebear-cyber/Number_Gussing_Game.git
cd Number_Gussing_Game
python number_guessing_game.py
```

- Python 3.x
